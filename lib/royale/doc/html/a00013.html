Detailed Description

This structure defines the depth data which is delivered through the callback.

This data comprises a dense 3D point cloud with the size of the depth image (width, height). The points vector encodes an array (row-based) with the size of (width x height). Based on the depthConfidence, the user can decide to use the 3D point or not.

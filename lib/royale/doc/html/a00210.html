enum RoyaleDotNet.CallbackData
strong

Specifies the type of data which should be captured and returned as callback.

Enumerator
None 

only get the callback but no data delivery

Raw 

raw frames, if exclusively used no processing pipe is executed (no calibration data is needed)

Depth 

one depth and grayscale image will be delivered for the complete sequence

Intermediate 

all intermediate data will be delivered which are generated in the processing pipeline

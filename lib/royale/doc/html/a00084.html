Detailed Description

template<class Category, class T, class Distance = ptrdiff_t, class Pointer = T *, class Reference = T &>
class royale::iterator::royale_iterator< Category, T, Distance, Pointer, Reference >

Royale non-const forward iterator class (used by royale containers)

Holding the specific functions for the non-const forward iterator. Other specific iterators shall not be based on this final iterator anymore.

The non-const forward iterator is used by the royale container classes and is compatible with the rest of the STL implementation concerning iterators

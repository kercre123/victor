thfAdaptEnrollmentQuickCheck

Syntax

#include <trulyhandsfree.h>
int thfAdaptEnrollmentQuickCheck(thf_t **thf, adaptObj_t *aObj, char *taskName, 
                                 short *wav, unsigned long wavLen, 
                                 unsigned short sampFreq, 
                                 thfEnrollmentCheck_t **enrollmentCheckResult);

Description

Performs a preliminary check on a waveform. Results are not as complete or necessarily as accurate as thfAdaptEnrollmentCheck, but many initial potential problems can be checked for as soon as the waveform is available. The sampling frequency of the waveform must be greater than or equal to the sampling frequency associated with the task.

The results are provided in a result object, described by the thfEnrollmentCheck_t structure. Information includes whether the waveform passed all checks (a binary indicator), and for each enrollment a list of specific checks, with the value and threshold for each check. For checks that are not applied to a single waveform (e.g. phrase quality, poor recordings, recording variance), the value is set equal to the threshold.

When done with the enrollment check result, it must be destroyed with a call to thfAdaptEnrollmentCheckDestroy

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
adaptObj_t * aObj in adapt object.
char * taskName in name of current task.
short * wav in pointer to audio recording of potential enrollment.
unsigned long wavLen in length of the input waveform, in samples.
unsigned short sampFreq in sample rate of input waveform (hz)
thfEnrollmentCheck_t ** enrollmentCheckResult out enrollment check object.

Return

Returns 1 on success or 0 on error.

Errors

String Description
adapt object not found A NULL pointer was passed in for the adapt object; pass in a correctly-initialized adapt object pointer created by thfAdaptCreate.
adapt object not initialized A non-NULL pointer was passed in for the adapt object, but this pointer does not contain a properly-initialized adapt object that is the output of thfAdaptCreate.
task name not specified A valid task name must be provided; a task with this name must have been created with a previous call to thfAdaptTaskCreate.
invalid waveform The input waveform is either NULL or has zero length; a valid waveform must be supplied.
result not specified A NULL pointer was passed in for the enrollment check object.
no result No enrollment check object could be obtained. Please contact customer support.
internal error: mismatch in trigger checks An internal error. Please contact customer support.
can't up-sample waveform The specified sampling frequency is less than the sampling frequency of the task. The waveforms used in enrollment must be sampled at no less than the sampling frequency of the task.

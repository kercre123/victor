thfUdtSaveToFile

Syntax

#include <trulyhandsfree.h>
int thfUdtSaveToFile(thf_t **thf, recog_t *r, searchs_t *r, char *recogFile, 
                     char *searchFile);

Description

Save a UDT search and recognizer to file. This function has the same type of usage as thfSpeakerSaveRecogToFile, but is used for user-defined triggers.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object, or NULL to not write any recognizer object.
searchs_t * r in Search object, or NULL to not write any search object.
char * recogFile in filename where recognizer object will be written
char * searchFile in filename where search object will be written

Return

Returns 1 on success or 0 on error.

Errors

String Description
failed to write search to file This function was unable to write the search object. A common cause is an invalid filename.
failed to write recognizer to file This function was unable to write the recognizer object. A common cause is an invalid filename.

thfSaveEmbedded

Syntax

#include <trulyhandsfree.h>
int thfSaveEmbedded(thf_t **thf, recog_t *r, searchs_t *s, const char *target, 
                    char *rfilename, char *sfilename, int reserved, 
                    int trigger);

Description

Saves recog and search objects to file using format compatible with specified embedded target. Contact Sensory for valid target string values.

In addition to the core recognizer and search, three parameters are written to the output file: (1) The value of REC_EPQ_MIN if REC_EPQ_ENABLE is non-zero, or an indication that EPQ is not used in this search. See thfRecogConfigSet to set these parameters. (2) PARAMA_OFFSET; see thfPhrasespotConfigSet to set this parameter. (3) Default DSP-level speaker-verification threshold; see thfSpeakerConfigSet to set this parameter.

Parameters

Type Name Dir. Description
thf_t * *thf in Session object.
recog_t * r in Recognizer object.
searchs_t * s in Search object.
const char * target in Target string.
char * rfilename in Recognizer filename
char * sfilename in Search filename.
int reserved in Reserved parameter. Use value 0.
int trigger in 1 if the search is continuously listening, or 0 if it is windowed.

Return

Returns 1 on success or 0 on error.

Errors

String Description
not a phrasespot search Search object must be configured for phrasespotting
embedded target "target" is not supported The string passed in via the target argument is not recognized as a valid embedded target

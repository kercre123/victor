Pipelined Text-to-Speech Synthesis

Pipelined synthesis generates as a series of small audio blocks (frames) on demand. This approach allows greater control than full synthesis. It uses less memory and allows for audio playback to begin before synthesis is complete. Pipelined synthesis steps are as follows :-
  1. Load a voice database using thfTtsCreateFromFile or thfTtsCreateFromStatic.
  2. Prepare to synthesize input text using thfTtsPrep. This returns the number of audio frames available and frame size (i.e., size of buffer in samples). Input parameters are used to configure the pitch and speech rate.
  3. Call thfTtsGetFrame to synthesize a frame of 16kHz 16-bit linear PCM audio. Call function repeatedly until all frames have been synthesized.
  4. Call thfFree to free each frame of audio.
  5. Call thfTtsReset before re-synthesizing a new phrase.
  6. Call thfTtsDestroy when done synthesizing to clean up memory.
